Mon paragraphe

1. Create a button
2. Add onclick attribute to the button. Here we want to execute the function changeColor()
3. Add parameter (color) on each button
4. Write the Javascript script
5. Declare our variables with let, var, or const, the one with which things are safer
6. Select the element we want to manipulate (getElementById(), getElementsByClassName(), or querySelector() )
7. Pass color parameter to the function
